Description
===========
Provides integration for the FOWiz SMS gateway and Drupal using the SMS Framework API.
You will need to register at http://cloud.fowiz.com/ in order for this module to work.

* sms_fowiz.module - The core file for the module, this defines the SMS Framework's settings form and
  send functions as well as a callback for the gateway to send data back to drupal.

Installation
============
1. Place the module in your Drupal's module folder.
2. Enable the module, and then configure the gateway, entering your Username and Passcode from FOWiz.
